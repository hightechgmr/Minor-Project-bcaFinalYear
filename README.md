# Nostalgic Gaming Online

Nostalgic Gaming Online is a web-based gaming platform developed as a Final Year BCA Project (2023вЂ“2026).
The project is designed to bring simple and nostalgic browser games together on a single online platform where users can play, track scores, and enjoy a clean gaming experience.

The website currently includes:

* Tic-Tac-Toe
* Puzzle Game

The platform allows users to:

* Create accounts and log in
* Play games online
* View scorecards and match statistics
* Track wins, losses, and draws
* Experience a responsive and user-friendly interface

---

## Project Objective

The main objective of this project is to develop an interactive online gaming platform using web technologies while applying concepts of:

* Frontend Development
* Backend Development
* Database Management
* User Authentication
* Responsive Web Design

This project also aims to improve teamwork, problem-solving, and practical implementation skills in full-stack web development.

## Installation & Setup

1. Install XAMPP or any local server environment.
2. Copy the project folder into the `htdocs` directory.
3. Start:

   * Apache
   * MySQL
4. Import the database file into phpMyAdmin.
5. Open your browser and run:

```text
http://localhost/project-folder-name
```
